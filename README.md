## Описание проекта

Приложение состоит из карты, строки поиска и виджетов. Поиск погоды осуществляется по названию населенного пункта или 
по координатам.

Виджеты:

- Яндек погода
- Accu Weather
- Open Weather

Логика общения с апи погоды в серверной части.
